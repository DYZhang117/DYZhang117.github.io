annotation

  • Dataset description:

The iris data set is a well-known classification dataset. I download this dataset from kaggle. This dataset is composed of measurement data of 3 kinds of iris:

Iris Setosa
Iris Versicolour
Iris Virginica

and each kind of iris contain 50 samples. Meanwhile, each row of data contains four attributes-Sepal.Length, Sepal.Width, Petal.Length, and Petal.Width.

switch

Five diagrams are provided. Change what is displayed based on the provided. For example, “sepalLength & sepalWidth” will be shown when the value of x is “sepalLength” and y is “sepalWidth”.
